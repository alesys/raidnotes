# Raid Notes

Raid Notes for the **Method Raid Tools** (MRT) addon for World of Warcraft.

## Description

This repository contains raid note configurations that can be imported directly into the Method Raid Tools addon. These notes help coordinate raid mechanics, assignments, and strategies.

## Usage

1. Copy the contents of the desired note file
2. Open Method Raid Tools in-game
3. Navigate to the Note section
4. Click "Import" and paste the note content
