# Raid Notes

Raid Notes for the **Method Raid Tools** (MRT) addon.

## Project Ascension - Manaforge Omega

This repository contains raid notes for the **Manaforge Omega** raid on Project Ascension (or similar custom server).
The notes are formatted for use with **Liquid's WeakAuras** pack.

## Bosses

### Dimensius the All-Devouring
- **File**: `dimensius.txt`
- **Mechanics**:
  - **Mass**: Likely Mass Dispel or Mass Disintegration.
  - **Gravity**: Gravity Well or similar pull/push mechanic.
  - **Crushing**: Crushing Shadows or similar.
  - **Nova**: Chaos Nova or expanding ring.
  - **CC**: Assignments for Crowd Control (Chaos Nova, Shockwave, Shadowfury).
  - **Bolas**: Void Bolas (balls/orbs).

## Usage

1. Copy the contents of the desired note file (e.g., `dimensius.txt`).
2. Open Method Raid Tools in-game.
3. Navigate to the Note section.
4. Click "Import" and paste the note content.
